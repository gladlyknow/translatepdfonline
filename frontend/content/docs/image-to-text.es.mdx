## Descripción general

**Imagen a Texto** extrae texto de cualquier imagen usando OCR con IA. Sube una foto, captura de pantalla o página escaneada y obtén texto editable en segundos. Exporta a TXT, DOCX, HTML o Markdown, y traduce o refina en el editor del banco de trabajo.

## Formatos de imagen compatibles

| Formato | Notas |
|---------|-------|
| **PNG** | Ideal para capturas de pantalla y gráficos |
| **JPG / JPEG** | Ideal para fotos |
| **BMP** | Sin compresión, calidad completa |
| **WebP** | Formato web moderno, archivos más pequeños |

Tamaño máximo: **50 MB**. Para resultados óptimos, usa imágenes claras y bien iluminadas donde el texto sea legible.

## Cómo extraer texto de una imagen

1. Ve a la página **[Imagen a Texto](/image-to-text)**.
2. Sube una imagen — arrastra y suelta, o haz clic para buscar.
3. Elige un **idioma objetivo** si planeas traducir después de la extracción.
4. Haz clic en **Imagen a Texto** — el OCR con IA procesa la imagen.
5. El **banco de trabajo OCR** se abre con el texto extraído.
6. Revisa, edita, traduce si es necesario y exporta.

## Casos de uso comunes

- **OCR de fotos** — toma una foto de un documento, pizarra, recibo o letrero y extrae su texto
- **OCR de capturas de pantalla** — extrae texto de capturas de IU, publicaciones en redes sociales, mensajes de error o registros de chat
- **OCR de documentos escaneados** — digitaliza una sola página escaneada a formato editable

## Formatos de salida

| Formato | Ideal para |
|---------|------------|
| **TXT** | Texto plano, copiar y pegar, edición simple |
| **DOCX** | Microsoft Word, documentos formateados |
| **HTML** | Publicación web, texto enriquecido |
| **Markdown** | Documentación, apps de notas, compatible con código |

## Idiomas compatibles

El OCR con IA detecta automáticamente el idioma del texto en la imagen. Idiomas soportados: inglés, chino, español, francés, alemán, japonés, coreano, ruso, griego, italiano y más. Tras la extracción, traduce el texto a cualquier idioma compatible desde el banco de trabajo.

## Consejos para mejores resultados

- Usa **imágenes nítidas y bien iluminadas** — evita el desenfoque o sombras intensas
- El texto debe estar **aproximadamente horizontal** y ser legible a tamaño de pantalla
- Para texto manuscrito, la escritura clara tipo imprenta produce mejores resultados que la cursiva
- Recorta al **área de texto** cuando sea posible para reducir el ruido de procesamiento
