## Overview

**PDF to Text** extracts text from any PDF using AI-powered OCR. Unlike PDF Translate — which translates text while preserving layout — PDF to Text focuses on getting editable text out of PDFs, especially scanned or image-heavy documents where text cannot be selected.

## Supported PDF types

- **Digital PDFs** — native text layers, extraction is fast and accurate
- **Scanned / image-based PDFs** — pages are images; AI OCR recognizes text from the visuals
- **Mixed content** — documents with both digital text and embedded images

Maximum file size: **50 MB**.

## How to extract text from a PDF

1. Go to the **[PDF to Text](/pdf-to-text)** page.
2. Upload a PDF — drag and drop, or click to browse.
3. Optionally pick a **target language** if you plan to translate after extraction.
4. Click **PDF to Text** — the AI OCR pipeline starts.
5. Wait for processing — the **OCR workbench** opens automatically with live progress.
6. Review and edit the extracted text in the workbench editor.
7. Export to your preferred format.

## Output formats

| Format | Best for |
|--------|----------|
| **TXT** | Plain text, copy-paste, simple editing |
| **DOCX** | Microsoft Word, formatted documents |
| **HTML** | Web publishing, rich text |
| **Markdown** | Documentation, note-taking apps, code-friendly |

## Language support

AI OCR detects the document language automatically. Supported languages include English, Chinese, Spanish, French, German, Japanese, Korean, Russian, Greek, Italian, and more. After extraction, the built-in translation engine can translate the text into any of these languages directly in the workbench.

## Tips for best results

- Use **clear, high-resolution scans** — 150 DPI or higher works best
- Ensure text is **legible and well-lit** in the source document
- For mixed-language documents, the AI auto-detects the dominant language
- If an export format is missing a specific feature, try exporting to **HTML** for maximum fidelity
