## 개요

**PDF 텍스트 변환**은 AI OCR을 사용하여 모든 PDF에서 텍스트를 추출합니다. 레이아웃을 유지하며 번역하는 PDF Translate와 달리, PDF 텍스트 변환은 특히 텍스트를 선택할 수 없는 스캔 문서나 이미지 기반 PDF에서 편집 가능한 텍스트를 얻는 데 중점을 둡니다.

## 지원 PDF 유형

- **디지털 PDF** — 네이티브 텍스트 레이어, 빠르고 정확한 추출
- **스캔 / 이미지 PDF** — 페이지가 이미지인 경우. AI OCR이 시각적으로 텍스트 인식
- **혼합 콘텐츠** — 디지털 텍스트와 포함된 이미지가 함께 있는 문서

최대 파일 크기: **50MB**.

## PDF에서 텍스트 추출하기

1. **[PDF 텍스트 변환](/pdf-to-text)** 페이지로 이동.
2. PDF 업로드 — 드래그 앤 드롭 또는 클릭하여 파일 선택.
3. 추출 후 번역할 경우 **대상 언어** 선택 (선택 사항).
4. **PDF 텍스트 변환** 클릭 — AI OCR 파이프라인 시작.
5. 처리 대기 — **OCR 워크벤치**가 자동으로 열리고 실시간 진행률 표시.
6. 워크벤치 편집기에서 추출된 텍스트 검토 및 편집.
7. 원하는 형식으로 내보내기.

## 출력 형식

| 형식 | 적합한 용도 |
|------|-----------|
| **TXT** | 일반 텍스트, 복사-붙여넣기, 간단한 편집 |
| **DOCX** | Microsoft Word, 서식 있는 문서 |
| **HTML** | 웹 게시, 리치 텍스트 |
| **Markdown** | 문서화, 노트 앱, 코드 친화적 |

## 언어 지원

AI OCR이 문서 언어를 자동으로 감지합니다. 지원 언어: 영어, 중국어, 스페인어, 프랑스어, 독일어, 일본어, 한국어, 러시아어, 그리스어, 이탈리아어 등. 추출 후 내장 번역 엔진으로 워크벤치에서 직접 지원되는 모든 언어로 번역할 수 있습니다.

## 가격 및 크레딧

신규 사용자는 PDF 텍스트 변환을 사용해 볼 수 있는 **무료 체험 크레딧**을 받습니다. 페이지당 현재 가격 설정에 따라 크레딧이 소비됩니다. 프리미엄 옵션에는 대용량 문서를 위한 고해상도 처리와 우선 처리 속도가 포함됩니다.

현재 크레딧 팩과 구독 플랜은 **[가격 페이지](/pricing)**에서 확인하세요.

## 최상의 결과를 위한 팁

- **선명한 고해상도 스캔** 사용 — 150 DPI 이상이 가장 좋은 결과를 제공
- 원본 문서의 텍스트가 **읽을 수 있고 균일하게 조명**되었는지 확인
- 다국어 문서의 경우 AI가 자동으로 주된 언어 감지
- 내보내기 형식에 특정 기능이 부족한 경우 최대 충실도를 위해 **HTML**로 내보내기 시도
