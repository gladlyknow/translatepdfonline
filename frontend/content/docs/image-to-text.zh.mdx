## 概述

**图片转文字** 使用 AI OCR 从任何图片中提取可编辑文字。上传照片、截图或扫描页，数秒内即可获得文字。导出为 TXT、DOCX、HTML 或 Markdown，在工作台编辑器中翻译或精修。

## 支持的图片格式

| 格式 | 说明 |
|------|------|
| **PNG** | 最适合截图和图形 |
| **JPG / JPEG** | 最适合照片 |
| **BMP** | 无压缩，全质量 |
| **WebP** | 现代网页格式，文件更小 |

文件上限：**50 MB**。为获得最佳效果，请使用清晰明亮、文字可读的图片。

## 如何从图片提取文字

1. 打开 **[图片转文字](/image-to-text)** 页面。
2. 上传图片——拖放或点击选择。
3. 如需在提取后翻译，选择**目标语言**。
4. 点击 **图片转文字**——AI OCR 开始处理图片。
5. **OCR 工作台**自动打开，显示提取的文字。
6. 校对、编辑，需要时翻译，然后导出。

## 常见使用场景

- **照片 OCR** — 拍摄文档、白板、收据或招牌，提取其中的文字
- **截图 OCR** — 从 UI 截图、社交媒体帖子、报错信息或聊天记录中提取文字
- **扫描件 OCR** — 将单页扫描件数字化为可编辑格式

## 输出格式

| 格式 | 适用场景 |
|------|----------|
| **TXT** | 纯文本，复制粘贴，简单编辑 |
| **DOCX** | Microsoft Word 文档，带格式排版 |
| **HTML** | 网页发布，富文本 |
| **Markdown** | 技术文档，笔记应用，代码友好 |

## 语言支持

AI OCR 自动检测图片中文字的语言。支持语言：中文、英文、西班牙语、法语、德语、日语、韩语、俄语、希腊语、意大利语等。提取完成后，可在工作台中将文字翻译为任意支持语言。

## 最佳效果提示

- 使用**光线充足、画面清晰**的图片——避免模糊或严重阴影
- 文字应**大致水平**，在屏幕尺寸下可读
- 手写文字以清晰的印刷体效果更佳，潦草字迹识别率较低
- 尽量**裁剪到文字区域**以减少处理噪音
