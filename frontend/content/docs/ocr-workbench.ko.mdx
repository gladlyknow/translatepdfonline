## OCR 사용 시점

PDF가 이미지/스캔 기반이라 일반 텍스트 추출이 어려울 때 OCR을 사용합니다.

## OCR 흐름

1. `/ocrtranslator` 에서 업로드된 파일 열기
2. 소스/타깃 언어 확인
3. OCR 작업 시작 후 완료 대기
4. 결과 확인 및 내보내기

## UX 규칙

- 문서/작업 컨텍스트가 있을 때는 재업로드가 비활성화됩니다.
- Translate가 스캔 PDF를 고신뢰로 감지하면 OCR로 안내합니다.
