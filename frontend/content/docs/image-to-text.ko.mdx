## 개요

**이미지 텍스트 변환**은 AI OCR을 사용해 모든 이미지에서 텍스트를 추출합니다. 사진, 스크린샷, 스캔 페이지를 업로드하면 몇 초 만에 편집 가능한 텍스트를 얻을 수 있습니다. TXT, DOCX, HTML, Markdown으로 내보내고 워크벤치 편집기에서 번역하거나 다듬을 수 있습니다.

## 지원 이미지 형식

| 형식 | 비고 |
|------|------|
| **PNG** | 스크린샷과 그래픽에 최적 |
| **JPG / JPEG** | 사진에 최적 |
| **BMP** | 비압축, 최고 품질 |
| **WebP** | 최신 웹 형식, 더 작은 파일 |

최대 파일 크기: **50MB**. 최상의 결과를 위해 텍스트가 읽을 수 있는 선명하고 밝은 이미지를 사용하세요.

## 이미지에서 텍스트 추출하기

1. **[이미지 텍스트 변환](/image-to-text)** 페이지로 이동.
2. 이미지 업로드 — 드래그 앤 드롭 또는 클릭하여 파일 선택.
3. 추출 후 번역하려면 **대상 언어** 선택.
4. **이미지 텍스트 변환** 클릭 — AI OCR이 이미지 처리.
5. **OCR 워크벤치**가 추출된 텍스트와 함께 열림.
6. 검토, 편집, 필요시 번역 후 내보내기.

## 주요 사용 사례

- **사진 OCR** — 문서, 화이트보드, 영수증, 표지판을 촬영하여 텍스트 추출
- **스크린샷 OCR** — UI 스크린샷, SNS 게시물, 오류 메시지, 채팅 기록에서 텍스트 추출
- **스캔 문서 OCR** — 단일 스캔 페이지를 편집 가능한 형식으로 디지털화

## 출력 형식

| 형식 | 적합한 용도 |
|------|-----------|
| **TXT** | 일반 텍스트, 복사-붙여넣기, 간단한 편집 |
| **DOCX** | Microsoft Word, 서식 있는 문서 |
| **HTML** | 웹 게시, 리치 텍스트 |
| **Markdown** | 문서화, 노트 앱, 코드 친화적 |

## 언어 지원

AI OCR이 이미지 내 텍스트 언어를 자동 감지합니다. 지원 언어: 영어, 중국어, 스페인어, 프랑스어, 독일어, 일본어, 한국어, 러시아어, 그리스어, 이탈리아어 등. 추출 후 워크벤치에서 지원되는 모든 언어로 번역할 수 있습니다.

## 가격 및 크레딧

신규 사용자에게 무료 체험 크레딧이 제공됩니다. 처리된 각 이미지는 현재 가격 설정에 따라 크레딧을 소비합니다. 프리미엄 처리는 더 높은 해상도와 우선 속도를 제공합니다.

자세한 내용은 **[가격 페이지](/pricing)**에서 확인하세요.

## 최상의 결과를 위한 팁

- **밝고 선명한 이미지** 사용 — 흐림이나 짙은 그림자 피하기
- 텍스트는 **대략 수평**이고 화면 크기에서 읽을 수 있어야 함
- 필기체의 경우 흘림체보다 또박또박 쓴 인쇄체가 더 나은 결과 제공
- 가능하면 **텍스트 영역**으로 자르기하여 처리 노이즈 감소
