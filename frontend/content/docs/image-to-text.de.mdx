## Übersicht

**Bild in Text** extrahiert Text aus jedem Bild mithilfe von KI-OCR. Laden Sie ein Foto, einen Screenshot oder eine gescannte Seite hoch und erhalten Sie in Sekunden bearbeitbaren Text. Exportieren Sie als TXT, DOCX, HTML oder Markdown und übersetzen oder verfeinern Sie das Ergebnis im Workbench-Editor.

## Unterstützte Bildformate

| Format | Hinweise |
|--------|----------|
| **PNG** | Am besten für Screenshots und Grafiken |
| **JPG / JPEG** | Am besten für Fotos |
| **BMP** | Unkomprimiert, volle Qualität |
| **WebP** | Modernes Webformat, kleinere Dateien |

Maximale Dateigröße: **50 MB**. Für optimale Ergebnisse verwenden Sie klare, gut ausgeleuchtete Bilder mit lesbarem Text.

## Text aus einem Bild extrahieren

1. Öffnen Sie die Seite **[Bild in Text](/image-to-text)**.
2. Laden Sie ein Bild hoch — per Drag-and-Drop oder Klick zum Durchsuchen.
3. Wählen Sie eine **Zielsprache**, falls Sie nach der Extraktion übersetzen möchten.
4. Klicken Sie auf **Bild in Text** — die KI-OCR verarbeitet das Bild.
5. Die **OCR-Workbench** öffnet sich mit dem extrahierten Text.
6. Überprüfen, bearbeiten, bei Bedarf übersetzen und exportieren.

## Häufige Anwendungsfälle

- **Foto-OCR** — machen Sie ein Foto von einem Dokument, Whiteboard, Beleg oder Schild und extrahieren Sie den Text
- **Screenshot-OCR** — extrahieren Sie Text aus UI-Screenshots, Social-Media-Posts, Fehlermeldungen oder Chat-Verläufen
- **OCR für gescannte Dokumente** — digitalisieren Sie eine einzelne gescannte Seite in ein editierbares Format

## Ausgabeformate

| Format | Am besten für |
|--------|---------------|
| **TXT** | Plaintext, Kopieren & Einfügen, einfache Bearbeitung |
| **DOCX** | Microsoft Word, formatierte Dokumente |
| **HTML** | Web-Publishing, Rich Text |
| **Markdown** | Dokumentation, Notiz-Apps, code-freundlich |

## Sprachunterstützung

KI-OCR erkennt die Textsprache im Bild automatisch. Unterstützte Sprachen: Englisch, Chinesisch, Spanisch, Französisch, Deutsch, Japanisch, Koreanisch, Russisch, Griechisch, Italienisch und mehr. Nach der Extraktion können Sie den Text aus der Workbench in jede unterstützte Sprache übersetzen.

## Tipps für beste Ergebnisse

- Verwenden Sie **gut ausgeleuchtete, scharfe Bilder** — vermeiden Sie Unschärfe oder starke Schatten
- Der Text sollte **etwa waagerecht** und in Bildschirmgröße lesbar sein
- Bei Handschrift liefert ein klarer Druckschriftstil bessere Ergebnisse als Schreibschrift
- Schneiden Sie nach Möglichkeit auf den **Textbereich** zu, um Verarbeitungsrauschen zu reduzieren
