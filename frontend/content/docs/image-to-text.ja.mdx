## 概要

**画像から文字を抽出** は、AI OCRを使ってあらゆる画像からテキストを取り出します。写真、スクリーンショット、スキャンページをアップロードすれば、数秒で編集可能なテキストが得られます。TXT、DOCX、HTML、Markdownでエクスポートし、ワークベンチエディタで翻訳や調整が可能です。

## 対応画像形式

| 形式 | 備考 |
|------|------|
| **PNG** | スクリーンショットや図形に最適 |
| **JPG / JPEG** | 写真に最適 |
| **BMP** | 非圧縮、フル品質 |
| **WebP** | 最新のWeb形式、ファイルサイズ小 |

最大ファイルサイズ: **50MB**。最適な結果を得るには、文字が判読できる鮮明で明るい画像を使用してください。

## 画像から文字を抽出する手順

1. **[画像から文字を抽出](/image-to-text)** ページを開く。
2. 画像をアップロード——ドラッグ＆ドロップ、またはクリックしてファイル選択。
3. 抽出後に翻訳する場合は**目標言語**を選択。
4. **画像から文字を抽出** をクリック——AI OCRが画像を処理。
5. **OCRワークベンチ**が開き、抽出された文字が表示される。
6. 確認・編集し、必要に応じて翻訳してエクスポート。

## よくある利用シーン

- **写真OCR** — 文書、ホワイトボード、レシート、看板を撮影して文字を抽出
- **スクリーンショットOCR** — UI画面、SNS投稿、エラーメッセージ、チャット履歴から文字を取り出す
- **スキャン文書OCR** — 単一ページのスキャンを編集可能な形式にデジタル化

## 出力フォーマット

| フォーマット | 最適な用途 |
|-------------|-----------|
| **TXT** | プレーンテキスト、コピペ、簡易編集 |
| **DOCX** | Microsoft Word、フォーマット付き文書 |
| **HTML** | Web公開、リッチテキスト |
| **Markdown** | 技術文書、ノートアプリ、コードとの親和性 |

## 言語サポート

AI OCRが画像内の文字言語を自動検出します。対応言語: 英語、中国語、スペイン語、フランス語、ドイツ語、日本語、韓国語、ロシア語、ギリシャ語、イタリア語他。抽出後、ワークベンチから任意の対応言語に翻訳できます。

## 最適な結果を得るためのヒント

- **明るく鮮明な画像**を使用——ぼやけや強い影を避ける
- 文字は**おおむね水平**で、画面サイズで判読できること
- 手書き文字は、筆記体より楷書体の方が認識精度が高い
- 可能であれば**文字領域**にトリミングしてノイズを低減
