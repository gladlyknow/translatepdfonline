## Aperçu

**Image en Texte** extrait le texte de toute image grâce à l'OCR IA. Téléversez une photo, une capture d'écran ou une page numérisée et obtenez du texte éditable en quelques secondes. Exportez en TXT, DOCX, HTML ou Markdown, puis traduisez ou peaufinez dans l'éditeur de l'atelier.

## Formats d'image pris en charge

| Format | Notes |
|--------|-------|
| **PNG** | Idéal pour les captures d'écran et les graphiques |
| **JPG / JPEG** | Idéal pour les photos |
| **BMP** | Sans compression, qualité maximale |
| **WebP** | Format web moderne, fichiers plus légers |

Taille maximale : **50 Mo**. Pour des résultats optimaux, utilisez des images nettes et bien éclairées où le texte est lisible.

## Comment extraire du texte d'une image

1. Allez sur la page **[Image en Texte](/image-to-text)**.
2. Téléversez une image — glissez-déposez ou cliquez pour parcourir.
3. Choisissez une **langue cible** si vous prévoyez de traduire après extraction.
4. Cliquez sur **Image en Texte** — l'OCR IA traite l'image.
5. L'**atelier OCR** s'ouvre avec le texte extrait.
6. Révisez, modifiez, traduisez si nécessaire, puis exportez.

## Cas d'usage courants

- **OCR photo** — prenez une photo d'un document, tableau blanc, reçu ou panneau et extrayez son texte
- **OCR capture d'écran** — extrayez le texte de captures d'interface, publications réseaux sociaux, messages d'erreur ou historiques de chat
- **OCR document numérisé** — numérisez une page scannée en format éditable

## Formats de sortie

| Format | Idéal pour |
|--------|------------|
| **TXT** | Texte brut, copier-coller, édition simple |
| **DOCX** | Microsoft Word, documents formatés |
| **HTML** | Publication web, texte enrichi |
| **Markdown** | Documentation, applications de notes, compatible code |

## Langues prises en charge

L'OCR IA détecte automatiquement la langue du texte dans l'image. Langues prises en charge : anglais, chinois, espagnol, français, allemand, japonais, coréen, russe, grec, italien et plus. Après extraction, traduisez le texte dans n'importe quelle langue prise en charge depuis l'atelier.

## Conseils pour de meilleurs résultats

- Utilisez des **images nettes et bien éclairées** — évitez le flou ou les ombres marquées
- Le texte doit être **à peu près horizontal** et lisible à la taille d'écran
- Pour l'écriture manuscrite, un style d'imprimerie clair donne de meilleurs résultats que l'écriture cursive
- Recadrez si possible sur la **zone de texte** pour réduire le bruit de traitement
