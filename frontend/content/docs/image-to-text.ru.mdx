## Обзор

**Изображение в Текст** извлекает текст из любого изображения с помощью ИИ OCR. Загрузите фотографию, скриншот или отсканированную страницу и получите редактируемый текст за считанные секунды. Экспортируйте в TXT, DOCX, HTML или Markdown, затем переводите или дорабатывайте в редакторе рабочей области.

## Поддерживаемые форматы изображений

| Формат | Примечания |
|--------|------------|
| **PNG** | Лучше всего для скриншотов и графики |
| **JPG / JPEG** | Лучше всего для фотографий |
| **BMP** | Без сжатия, полное качество |
| **WebP** | Современный веб-формат, меньший размер файла |

Максимальный размер файла: **50 МБ**. Для оптимальных результатов используйте чёткие, хорошо освещённые изображения с читаемым текстом.

## Как извлечь текст из изображения

1. Перейдите на страницу **[Изображение в Текст](/image-to-text)**.
2. Загрузите изображение — перетащите или нажмите для выбора файла.
3. Выберите **целевой язык**, если планируете перевод после извлечения.
4. Нажмите **Изображение в Текст** — ИИ OCR обрабатывает изображение.
5. Открывается **рабочая область OCR** с извлечённым текстом.
6. Проверьте, отредактируйте, при необходимости переведите и экспортируйте.

## Типичные сценарии использования

- **OCR фото** — сфотографируйте документ, доску, чек или вывеску и извлеките текст
- **OCR скриншотов** — извлекайте текст из скриншотов интерфейса, постов в соцсетях, сообщений об ошибках или истории чатов
- **OCR отсканированных документов** — оцифруйте отдельную отсканированную страницу в редактируемый формат

## Форматы вывода

| Формат | Лучше всего для |
|--------|-----------------|
| **TXT** | Простой текст, копирование-вставка, простое редактирование |
| **DOCX** | Microsoft Word, форматированные документы |
| **HTML** | Веб-публикация, форматированный текст |
| **Markdown** | Документация, приложения для заметок, совместимость с кодом |

## Поддержка языков

ИИ OCR автоматически определяет язык текста на изображении. Поддерживаемые языки: английский, китайский, испанский, французский, немецкий, японский, корейский, русский, греческий, итальянский и другие. После извлечения переведите текст на любой поддерживаемый язык прямо из рабочей области.

## Советы для лучших результатов

- Используйте **хорошо освещённые, чёткие изображения** — избегайте размытия и сильных теней
- Текст должен быть **примерно горизонтальным** и читаемым при размере экрана
- Для рукописного текста чёткий печатный стиль даёт лучшие результаты, чем прописной
- По возможности обрезайте до **текстовой области**, чтобы уменьшить шум обработки
