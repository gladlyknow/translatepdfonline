## Cargar e iniciar tareas

En `/translate/upload`:

1. Carga un archivo PDF.
2. Selecciona idioma de origen y destino.
3. Inicia:
   - **PDF Translate** para PDF con texto.
   - **PDF OCR** para PDF escaneados o de imagen.

## Notas

- El historial muestra tareas y archivos cargados.
- Si el archivo se detecta como escaneado con alta confianza, Translate se bloquea y se recomienda OCR.
- Se eliminaron accesos redundantes tras la carga para mantener un único flujo de inicio.
