## Upload and start tasks

On `/translate/upload`, complete this sequence:

1. Upload a PDF file.
2. Select **Source Language** and **Target Language**.
3. Start either:
   - **PDF Translate** for text-based PDFs
   - **PDF OCR** for scanned/image PDFs

## Notes

- The system keeps task history and uploaded files in the History drawer.
- If a file is detected as a high-confidence scanned PDF, Translate is blocked and OCR is recommended.
- Redundant post-upload shortcut buttons are removed to keep one clear start flow.
