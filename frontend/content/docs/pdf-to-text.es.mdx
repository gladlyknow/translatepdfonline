## Descripción general

**PDF a Texto** extrae texto de cualquier PDF usando OCR con IA. A diferencia de PDF Translate — que traduce preservando el diseño — PDF a Texto se enfoca en obtener texto editable de PDFs, especialmente documentos escaneados o basados en imágenes donde el texto no es seleccionable.

## Tipos de PDF compatibles

- **PDFs digitales** — capas de texto nativas, extracción rápida y precisa
- **PDFs escaneados / de imagen** — las páginas son imágenes; el OCR con IA reconoce el texto visualmente
- **Contenido mixto** — documentos con texto digital e imágenes incrustadas

Tamaño máximo: **50 MB**.

## Cómo extraer texto de un PDF

1. Ve a la página **[PDF a Texto](/pdf-to-text)**.
2. Sube un PDF — arrastra y suelta, o haz clic para buscar.
3. Opcionalmente elige un **idioma objetivo** si planeas traducir tras la extracción.
4. Haz clic en **PDF a Texto** — el OCR con IA comienza.
5. Espera el procesamiento — el **banco de trabajo OCR** se abre automáticamente con el progreso en vivo.
6. Revisa y edita el texto extraído en el editor.
7. Exporta al formato que prefieras.

## Formatos de salida

| Formato | Ideal para |
|---------|------------|
| **TXT** | Texto plano, copiar y pegar, edición simple |
| **DOCX** | Microsoft Word, documentos formateados |
| **HTML** | Publicación web, texto enriquecido |
| **Markdown** | Documentación, apps de notas, compatible con código |

## Idiomas compatibles

El OCR con IA detecta automáticamente el idioma del documento. Idiomas soportados: inglés, chino, español, francés, alemán, japonés, coreano, ruso, griego, italiano y más. Tras la extracción, el motor de traducción integrado puede traducir el texto a cualquiera de estos idiomas directamente en el banco de trabajo.

## Precios y créditos

Los nuevos usuarios reciben **créditos de prueba gratuitos** para probar PDF a Texto. Cada página consume créditos según la configuración de precios actual. Las opciones premium incluyen procesamiento de mayor resolución y velocidad prioritaria para documentos grandes.

Consulta la **[página de Precios](/pricing)** para los packs de créditos y planes de suscripción actuales.

## Consejos para mejores resultados

- Usa **escaneos claros y de alta resolución** — 150 DPI o más funciona mejor
- Asegúrate de que el texto sea **legible y esté bien iluminado** en el documento original
- Para documentos multilingües, la IA detecta automáticamente el idioma dominante
- Si un formato de exportación carece de alguna función, prueba exportar a **HTML** para máxima fidelidad
