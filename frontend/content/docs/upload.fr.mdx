## Televerser et demarrer une tache

Sur `/translate/upload` :

1. Televersez un PDF.
2. Choisissez langue source et langue cible.
3. Lancez :
   - **PDF Translate** pour les PDF avec texte extractible.
   - **PDF OCR** pour les PDF scannes/image.

## Notes

- L'historique affiche les taches et fichiers televerses.
- Si un PDF est detecte comme scan avec forte confiance, Translate est bloque et OCR est recommande.
- Les boutons redondants apres televersement sont retires pour garder un flux unique.
