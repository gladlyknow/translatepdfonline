## Overview

**Image to Text** extracts text from any image using AI-powered OCR. Upload a photo, screenshot, or scanned page and get editable text in seconds. Export to TXT, DOCX, HTML, or Markdown — then translate or refine in the workbench editor.

## Supported image formats

| Format | Notes |
|--------|-------|
| **PNG** | Best for screenshots and graphics |
| **JPG / JPEG** | Best for photos |
| **BMP** | Uncompressed, full quality |
| **WebP** | Modern web format, smaller files |

Maximum file size: **50 MB**. For optimal results, use clear, well-lit images where text is readable.

## How to extract text from an image

1. Go to the **[Image to Text](/image-to-text)** page.
2. Upload an image — drag and drop, or click to browse.
3. Pick a **target language** if you plan to translate after extraction.
4. Click **Image to Text** — AI OCR processes the image.
5. The **OCR workbench** opens with the extracted text.
6. Review, edit, translate if needed, then export.

## Common use cases

- **Photo OCR** — snap a picture of a document, whiteboard, receipt, or sign and extract its text
- **Screenshot OCR** — pull text from UI screenshots, social media posts, error messages, or chat logs
- **Scanned document OCR** — digitize a single scanned page into editable format

## Output formats

| Format | Best for |
|--------|----------|
| **TXT** | Plain text, copy-paste, simple editing |
| **DOCX** | Microsoft Word, formatted documents |
| **HTML** | Web publishing, rich text |
| **Markdown** | Documentation, note-taking apps, code-friendly |

## Language support

AI OCR auto-detects the text language in the image. Supported languages: English, Chinese, Spanish, French, German, Japanese, Korean, Russian, Greek, Italian, and more. After extraction, translate the text into any supported language from within the workbench.

## Tips for best results

- Use **well-lit, sharp images** — avoid blur or heavy shadows
- Text should be **roughly horizontal** and readable at screen size
- For handwriting, clear print-style writing produces better results than cursive
- Crop to the **text area** when possible to reduce processing noise
