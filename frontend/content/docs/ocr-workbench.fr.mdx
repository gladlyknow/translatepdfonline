## Quand utiliser OCR

Utilisez OCR pour les PDF scannes/image lorsque l'extraction de texte classique n'est pas fiable.

## Flux OCR

1. Ouvrez `/ocrtranslator` avec un fichier deja televerse.
2. Confirmez langue source et cible.
3. Lancez la tache OCR et attendez la fin.
4. Verifiez et exportez le resultat.

## Regles UX

- En presence d'un document/tache actif, le re-upload est desactive.
- Si Translate detecte un scan avec forte confiance, l'utilisateur est guide vers OCR.
