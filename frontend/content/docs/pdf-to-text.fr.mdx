## Aperçu

**PDF en Texte** extrait le texte de tout PDF grâce à l'OCR IA. Contrairement à PDF Translate — qui traduit en préservant la mise en page — PDF en Texte se concentre sur l'extraction de texte éditable, particulièrement pour les PDF numérisés ou basés sur des images où le texte n'est pas sélectionnable.

## Types de PDF pris en charge

- **PDF numériques** — couches de texte natives, extraction rapide et précise
- **PDF numérisés / image** — les pages sont des images ; l'OCR IA reconnaît le texte visuellement
- **Contenu mixte** — documents avec texte numérique et images intégrées

Taille maximale : **50 Mo**.

## Comment extraire du texte d'un PDF

1. Allez sur la page **[PDF en Texte](/pdf-to-text)**.
2. Téléversez un PDF — glissez-déposez ou cliquez pour parcourir.
3. Choisissez éventuellement une **langue cible** si vous prévoyez de traduire après extraction.
4. Cliquez sur **PDF en Texte** — le pipeline OCR IA démarre.
5. Attendez le traitement — l'**atelier OCR** s'ouvre automatiquement avec la progression en direct.
6. Révisez et modifiez le texte extrait dans l'éditeur.
7. Exportez dans le format de votre choix.

## Formats de sortie

| Format | Idéal pour |
|--------|------------|
| **TXT** | Texte brut, copier-coller, édition simple |
| **DOCX** | Microsoft Word, documents formatés |
| **HTML** | Publication web, texte enrichi |
| **Markdown** | Documentation, applications de notes, compatible code |

## Langues prises en charge

L'OCR IA détecte automatiquement la langue du document. Langues prises en charge : anglais, chinois, espagnol, français, allemand, japonais, coréen, russe, grec, italien et plus. Après extraction, le moteur de traduction intégré peut traduire le texte dans n'importe quelle langue prise en charge depuis l'atelier.

## Conseils pour de meilleurs résultats

- Utilisez des **numérisations nettes en haute résolution** — 150 DPI ou plus donne les meilleurs résultats
- Assurez-vous que le texte est **lisible et bien éclairé** dans le document source
- Pour les documents multilingues, l'IA détecte automatiquement la langue dominante
- Si un format d'export manque une fonctionnalité, essayez l'export en **HTML** pour une fidélité maximale
