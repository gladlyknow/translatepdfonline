## Τι είναι το Translate PDF Online;

Το **Translate PDF Online** είναι υπηρεσία ιστού για **μετάφραση PDF** με διατήρηση της διάταξης, σχημάτων και πινάκων όπου είναι εφικτό. Απευθύνεται σε ακαδημαϊκά και επαγγελματικά PDF σε **αγγλικά, απλοποιημένα κινεζικά, ισπανικά, γαλλικά, ιταλικά, ελληνικά, ιαπωνικά, κορεατικά**, με περισσότερα ζεύγη γλωσσών στο μέλλον.

- **Ιστότοπος:** [www.translatepdfonline.com](https://www.translatepdfonline.com)
- **Επικοινωνία:** [hello@translatepdfonline.com](mailto:hello@translatepdfonline.com)

## Ανοικτός κώδικας και αναγνωρίσεις

Η ροή μετάφρασης βασίζεται σε έργα ανοικτού κώδικα, ιδίως το **[zotero-pdf2zh](https://github.com/guaguastandup/zotero-pdf2zh)** και το οικοσύστημα **pdf2zh / BabelDOC**.

## Συμμόρφωση AGPL-3.0

Στοιχεία από το **pdf2zh** (AGPL-3.0) μπορεί να επιβάλλουν υποχρεώσεις παροχής πηγαίου κώδικα. **Αναφορά:** [github.com/guaguastandup/zotero-pdf2zh](https://github.com/guaguastandup/zotero-pdf2zh). Για το deployment μας: [hello@translatepdfonline.com](mailto:hello@translatepdfonline.com).

_Το τμήμα αυτό είναι ενημερωτικό, όχι νομική συμβουλή._

## Αρχιτεκτονική

**Ξεχωριστό front end και back end.**

### Front (edge)

- **Στοίβα:** React / **Next.js** σε **Cloudflare Pages / Workers**.
- **Ρόλος:** UI, ταυτοποίηση, χρεώσεις, προεπισκόπηση PDF, κλήσεις API.

### Αποθήκευση αντικειμένων

- **Cloudflare R2** με **S3**-συμβατά API.

### Backend API

- **Python (FastAPI)** σε cloud· λογαριασμοί, πιστώσεις, εργασίες, **DeepSeek**, ουρές.

### Ασύγχρονοι workers

- **Celery** (ή ισοδύναμο): ανάλυση διάταξης, εξαγωγή κειμένου, μετάφραση, **BabelDOC**.

**Ροή:** πρόγραμμα περιήγησης → Next.js (Cloudflare) → API → R2, ουρές, DeepSeek → workers → R2.

## Βασικές λειτουργίες

### A. Προβολή PDF και επιλογή περιοχής

- Επίπεδο αλληλεπίδρασης (π.χ. `react-pdf-selection`), κανονικοποιημένες συντεταγμένες, μερική μετάφραση με **PyMuPDF** και **DeepSeek**.

### B. Σύγκριση δίπλα-δίπλα

- **pdf.js**· συγχρονισμός κύλισης.

### C. Αγωγός εργασιών

1. Ανάλυση / διάταξη  
2. Μετάφραση (**DeepSeek**)  
3. Απόδοση (**BabelDOC**)

## Χώρος μετάφρασης (πίνακας Ιστορικό)

Στη σελίδα **[Μετάφραση PDF](/translate/upload)**, η στήλη **Ιστορικό** είναι κεντρική:

- Εμφανίζει **μεταφορτώσεις και εργασίες μετάφρασης** της συνεδρίας (και του **λογαριασμού** σας αν έχετε συνδεθεί): κατάσταση, γλώσσες, ονόματα αρχείων.
- **Ανοίξτε προηγούμενη εργασία** από το ιστορικό για προεπισκόπηση, λήψη μεταφρασμένου PDF ή παρακολούθηση προόδου **χωρίς να ανεβάσετε ξανά το ίδιο αρχείο**.
- Το ιστορικό συνδέεται με την **προεπισκόπηση PDF** και τα στοιχεία ελέγχου δεξιά.

Αν λείπει μια εγγραφή, μπορεί να διαγράφηκε από **πολιτική διατήρησης** ή να χρησιμοποιείτε άλλη συσκευή/συνεδρία.

## Ασφάλεια και διατήρηση

- **Μεταφορά:** **HTTPS** άκρο-προς-άκρο.
- **Διατήρηση:** αυτόματη διαγραφή αντικειμένων στο **R2** μετά περίπου **7 ημέρες**, εκτός αν ορίζεται αλλιώς.
- **Ασφάλεια:** λογικοί έλεγχοι πρόσβασης και κρυπτογράφηση κατά τη μεταφορά.
