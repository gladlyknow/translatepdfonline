## Was ist der Vertragsvergleich?

Der **Vertragsvergleich** ist ein KI-gestütztes Werkzeug, das zwei Versionen eines Dokuments analysiert und jede Abweichung markiert — hinzugefügter, gelöschter oder geänderter Text — damit Sie Änderungen prüfen können, ohne Seiten manuell Zeile für Zeile durchzugehen.

Er funktioniert für Verträge, rechtliche Vereinbarungen, Richtlinienentwürfe, AGB und jedes textbasierte PDF. Laden Sie ein Original und eine geänderte Fassung hoch; das Werkzeug liefert ein strukturiertes Diff mit Ähnlichkeitsbewertung und kategorisierten Änderungseinträgen.

## Funktionsweise

1. **Laden Sie** das Originaldokument und die geänderte Version hoch (beide müssen PDF sein).
2. Das System extrahiert den Text aus beiden Dateien und führt einen KI-Vergleich durch.
3. Die Ergebnisse zeigen einen **Ähnlichkeitsprozentsatz**, eine **Diff-Zusammenfassung** (Gesamteinträge, hinzugefügt, gelöscht, geändert) und strukturierte Details zu jeder Änderung.

Der Vergleich findet serverseitig statt. Große Dokumente können länger dauern; das Werkzeug zeigt den Fortschritt während der Ausführung an.

## Funktionen

- **Seite-an-Seite-Vergleich** — Original- und geänderter Text werden parallel angezeigt.
- **Ähnlichkeitswert** — ein Prozentwert, der angibt, wie nahe sich die beiden Dokumente sind.
- **Kategorisierte Unterschiede** — Änderungen gruppiert nach hinzugefügt, gelöscht oder geändert.
- **Auftragsverlauf** — frühere Vergleiche werden gespeichert und können über das Verlaufsfenster wieder geöffnet werden.
- **Teilbare Ergebnisse** — jeder Vergleich erhält eine eindeutige URL, die Sie mit Kollegen teilen können.

## Unterstützte Formate

- **Eingabe:** PDF (textbasiert; gescannte PDFs sollten zuerst durch OCR — siehe [OCR-Werkbank](/docs/ocr-workbench)).
- **Ausgabe:** Interaktive Diff-Ansicht im Browser; Rohdaten des Diffs sind über die Ergebnisseite zugänglich.

## Hinweise

- Die Vergleichsqualität hängt von der Genauigkeit der Textextraktion ab. Stark gescannte oder bildlastige PDFs können unvollständige Ergebnisse liefern.
- Das Werkzeug ist für textuelle Unterschiede in Verträgen und juristischen Dokumenten optimiert; es vergleicht kein visuelles Layout, keine Unterschriften oder Stempel.
- Hochgeladene Dateien unterliegen derselben [Aufbewahrungsrichtlinie](/docs#datensicherheit-und-aufbewahrung) wie andere Dokumente auf der Plattform.

Das vollständige interaktive Werkzeug finden Sie unter [Vertragsvergleich](/contract-comparison).
