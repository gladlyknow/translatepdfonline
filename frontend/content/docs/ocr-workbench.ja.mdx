## OCR を使う場面

PDF が画像ベースで通常のテキスト抽出が難しい場合は OCR を利用します。

## OCR フロー

1. `/ocrtranslator` でアップロード済みファイルを開く
2. ソース言語とターゲット言語を確認
3. OCR タスクを開始して完了を待つ
4. 結果を確認してエクスポート

## UX ルール

- ドキュメント/タスクのコンテキストがある場合、再アップロードは無効になります。
- Translate が高信頼でスキャン PDF を検知した場合、OCR へ誘導されます。
