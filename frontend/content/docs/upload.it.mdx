## Carica e avvia task

In `/translate/upload`:

1. Carica un file PDF.
2. Seleziona lingua sorgente e lingua target.
3. Avvia:
   - **PDF Translate** per PDF con testo.
   - **PDF OCR** per PDF scansionati o immagini.

## Note

- La cronologia mostra task recenti e file caricati.
- Se il file e rilevato come PDF scansionato ad alta confidenza, Translate viene bloccato e si consiglia OCR.
- I pulsanti ridondanti dopo upload sono stati rimossi per mantenere un unico flusso.
