## Загрузка и запуск задачи

На `/translate/upload`:

1. Загрузите PDF-файл.
2. Выберите исходный и целевой языки.
3. Запустите:
   - **PDF Translate** для PDF с извлекаемым текстом.
   - **PDF OCR** для сканов/изображений PDF.

## Примечания

- В истории показываются недавние задачи и загруженные файлы.
- Если файл с высокой уверенностью определен как скан, Translate блокируется и предлагается OCR.
- Лишние кнопки после загрузки удалены, чтобы оставить единый понятный стартовый поток.
