## 업로드 후 작업 시작

`/translate/upload` 에서 다음 순서로 진행합니다.

1. PDF 파일 업로드
2. 소스 언어와 타깃 언어 선택
3. 다음 중 하나 시작
   - **PDF Translate** (텍스트 기반 PDF)
   - **PDF OCR** (스캔/이미지 PDF)

## 참고

- 히스토리 드로어에서 최근 작업과 업로드 파일을 볼 수 있습니다.
- 파일이 높은 신뢰도로 스캔 PDF로 감지되면 Translate는 차단되고 OCR로 안내됩니다.
- 업로드 후 중복 이동 버튼은 제거되어 시작 흐름이 단순화됩니다.
