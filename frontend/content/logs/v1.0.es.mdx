**Translate PDF Online** está disponible en **www.translatepdfonline.com**: suba PDF académicos o profesionales, traduzca entre **inglés**, **chino (simplificado)** y **español** (y más idiomas con el tiempo) y descargue PDF traducidos conservando el diseño cuando el sistema lo permita.

## Aspectos destacados

- **Aplicación web** en **Cloudflare** (Next.js) con inicio de sesión y área de traducción  
- Almacenamiento **R2**; **HTTPS** en todo el tráfico  
- **Workers** asíncronos para análisis de maquetación (p. ej. DocLayout-YOLO) y re-render tipo **BabelDOC**  
- API **DeepSeek**; facturación por **créditos** por página traducida  
- Pipeline alineado con **[zotero-pdf2zh](https://github.com/guaguastandup/zotero-pdf2zh)** y el ecosistema **pdf2zh / BabelDOC**  

## Políticas

- **Privacidad** y **Términos** describen el uso de datos, retención de archivos de unos **7 días** en almacenamiento de objetos y usos prohibidos.  
- La transparencia **AGPL** para componentes upstream está en la **[Documentación](/docs)**.
