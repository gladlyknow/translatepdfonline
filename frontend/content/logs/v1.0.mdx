**Translate PDF Online** is now available at **www.translatepdfonline.com**: upload academic and professional PDFs, choose **English**, **Chinese (Simplified)**, or **Spanish** (and more over time), and download translated PDFs while preserving layout where supported.

## Highlights

- **Web app** on **Cloudflare** (Next.js) with sign-in and translation workspace  
- **R2** storage for uploads and outputs; **HTTPS** everywhere  
- **Asynchronous workers** for layout analysis (e.g. DocLayout-YOLO) and **BabelDOC**-style re-rendering  
- **DeepSeek** API for translation; **credit-based** billing per translated page  
- Pipeline informed by **[zotero-pdf2zh](https://github.com/guaguastandup/zotero-pdf2zh)** and the **pdf2zh / BabelDOC** ecosystem  

## Policies

- **Privacy** and **Terms** describe data use, **~7-day** file retention in object storage, and acceptable use.  
- **AGPL** transparency for upstream components is summarized in **[Documentation](/docs)**.
