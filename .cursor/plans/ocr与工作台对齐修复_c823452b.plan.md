# OCR / Header / Workbench 对齐实施方案

## 目标范围
- 修复 OCR 任务链路中的 `enqueue_runtime_error (Illegal invocation)`，并保证队列阶段可持续推进。
- 重构 OCR 的 `Hist & Log` 展示顺序与分页（Recent/Uploaded 各 10 条、可翻页）。
- 统一 `/upload` 顶部导航与首页一致（包含 `Upload`）。
- 参照 `D:/imppro/onlinepdftranslator` 补齐 OCR workbench 的字体/文本编辑按钮。
- 让 translate 左侧控制台与 OCR 对齐（`Home / Upload / Hist & Log`），并复用相同 Hist & Log 展示顺序。
- OCR 左侧控制台增加紧凑 `Credits / Buy credits` 模块，跳转到 `Pricing`，且不挤压核心操作区。
- OCR 的可视化 JSON 控件集固定展示（sticky/fixed 区域），不随 JSON 列表滚动；整体风格与站点一致。
- OCR 与 translate 页面不再显示 `Show header / Show footer` 浮层文案，默认隐藏页首/页尾控制提示。
- 在左侧控件集新增 `Pages` 控制组（`Expand header / Prev / Source Page / Next / Expand footer`），用于页首页尾展开隐藏，以及源文件与译文/JSON 相关翻页联动。
