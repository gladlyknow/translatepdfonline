# OCR 工作台终修执行计划

## 目标
- OCR 主区固定为双栏：左侧 Source PDF、右侧 JSON 可视化 workbench。
- 右侧不再出现纯文本 JSON 预览，也不再回退到 target PDF 面板。
- `Undo / Redo / Save / Export MD / PDF export` 仅保留在左侧 `File` 工具组，移除 workbench 顶部重复按钮。
- `PDF export` 生成完成后在 `File` 区显示 `Download` 按钮可直接下载。
- 删除 `Block props` 分组，`Text edit / Font settings / File` 三组按网站风格重排。
- 左侧工具集固定在页面最左侧（sticky/fixed），不随 source/workbench 内容滚动。
- OCR 轮询在 `completed` 后可稳定停止，消除 `/api/tasks/*` 与 `/api/ocr/tasks/*/exports` 持续高频请求。

## 变更步骤

### 1) 固定 OCR 主区结构（不再 fallback）
- 在 [D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx](D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx) 中移除 `source + target` fallback 渲染路径。
- 主区统一改为：
  - 左栏：`PdfViewerPane(mode="source")`
  - 右栏：`OcrParseWorkbench(hideSourcePanel, toolbarPosition="left")`
- `Pages` 焦点仅由点击 source/workbench 面板切换（保持默认 JSON）。

### 2) 顶部按钮去重，只保留左侧工具栏 File 区（含 PDF export/download）
- 在 [D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/OcrParseWorkbench.tsx](D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/OcrParseWorkbench.tsx) 增加条件：`showLeftToolbar` 时隐藏顶部 `Undo/Redo/Save/Export MD`。
- 保持 `fileControls` 继续传给左侧工具栏（`ParseResultEditorToolbar`），确保 `File` 区按钮完整可用。
- 在 `File` 组补齐 `PDF export` 按钮及导出完成后的 `Download` 按钮链路（状态：idle/processing/ready/failed）。
- 在 [D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/parse-result-editor-toolbar.tsx](D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/parse-result-editor-toolbar.tsx) 仅做样式微调，不改功能事件。

### 3) 左侧控件逐像素微调（视觉贴近截图）+ 固定最左工具集
- 在 [D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx](D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx) 统一按钮尺寸、分组标题层级、间距与 hover 强度。
- 在 [D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/OcrParseWorkbench.tsx](D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/OcrParseWorkbench.tsx) 与 [D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/parse-result-editor-toolbar.tsx](D:/imppro/translatepdfonline/frontend/src/shared/ocr-workbench/parse-result-editor-toolbar.tsx) 调整左侧工具栏卡片密度、圆角、边框与悬停反馈，保持站点风格一致。
- 删除 `Block props` 区块，仅保留 `Text edit / Font settings / File`，并把工具栏容器固定在页面最左侧，不随内容滚动。

### 4) 轮询状态机硬收敛（解决 completed 后仍刷新）
- 在 [D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx](D:/imppro/translatepdfonline/frontend/src/app/[locale]/(translate)/ocrtranslator/OcrTranslatePageClient.tsx) 重构 OCR 轮询为“单会话 + 单飞 + 可终止”机制：
  - `pollSessionId` 防旧请求回写。
  - `task/exports` 单飞锁，杜绝并发轮询。
  - `completed` 后按导出状态机判断终止：
    - 有 `ready` 立即停止。
    - 全终态（`ready/failed/cancelled`）但无 ready 也停止并提示可手动重试导出。
    - 仅 `pending/processing` 才继续轮询。
  - 增加错误退避和上限，避免异常时无限刷。
- 对 `task` 与 `exports` 结果加快照去重，状态未变化不触发重复 `setState`。

- 运行 `pnpm exec tsc --noEmit` 与改动文件 lint。
- 本地验证：
  - OCR 主区只剩 `Source + OCR workbench`。
  - 顶部不再出现 `Undo/Redo/Save/Export MD`。
  - 左侧 `File` 区按钮完整可用，含 `PDF export` 与导出完成后的 `Download`。
  - 左侧只保留 `Text edit / Font settings / File`，不再出现 `Block props`。
  - 左侧工具集固定在页面最左侧，滚动 source/workbench 内容时工具集不跟随滚动。
  - `completed + 导出 ready` 后，网络请求停止；不再持续出现 `/api/tasks/*` 与 `/api/ocr/tasks/*/exports` 周期请求。
